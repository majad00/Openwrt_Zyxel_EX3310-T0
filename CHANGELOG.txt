INSTALLATION NOTES
------------------
REQUIRED FILES:
   • Zyxel-Matrix-Loader.exe 
   • openwrt-zyxel-matrix-14.07-rtkmips-wsr30-squashfs-sysupgrade-v4.0 (v4.0.0)

COMPATIBILITY:
   • Router: Zyxel Multy U AC2100 (WSR30) only
   • OS: Windows 10/11 (with TFTP client enabled)
   • Browser: Chrome, Firefox, Edge (for Luci interface)

UPGRADE NOTES:
   • Always backup configuration before upgrading
   • Use same flashing procedure as initial install
   • Settings may need reconfiguration after upgrade
